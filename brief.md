## 1. ЦЕЛЬ И ПОЗИЦИОНИРОВАНИЕ

### Цель сайта
Конвертировать посетителя из «узнал» → «доверяю» → «записался на диагностическую сессию».

Единственное целевое действие: **написать репетитору в Telegram (@mila_konstanz)**.

### Позиционирование
Не «онлайн-курс немецкого» и не «репетитор по объявлению».

**«Тёплый наставник с живым присутствием в Германии, который использует AI-инструменты как часть персональной системы — и получает с учениками твёрдые, измеримые результаты.»**

Три кита позиционирования:
1. **Наставник, не учитель** — трансформация, не трансляция знаний
2. **Живёт в Германии** — знает реальный язык изнутри, не из учебника
3. **AI-усиленная система** — ежедневная практика между сессиями, не только еженедельный урок

### Уникальное торговое предложение (УТП)
> «Перестать переводить и начать думать по-немецки — с живым наставником из Германии и умной системой практики между занятиями.»

## 2. ЦЕЛЕВАЯ АУДИТОРИЯ

### Первичная аудитория: Русскоязычные жители DACH, уровень A2–B2

**Демография:**
- Возраст: 25–45 лет
- Происхождение: Россия, Украина, Казахстан, Беларусь
- Место жительства: Германия, Австрия, Швейцария (переехали или планируют)
- Профессия: IT-специалисты, инженеры, медики, студенты, предприниматели

**Психография — 5 главных сценариев:**

| Сценарий | Боль | Цель |
|---|---|---|
| Переезд/интеграция | «Понимаю, но в Behörde теряюсь» | Уверенная бытовая речь |
| Работа/карьера | «На встречах молчу — боюсь ошибиться» | Профессиональная коммуникация |
| Экзамен Goethe/telc/TestDaF | «Говорение — самое страшное» | Сдать с первого раза |
| Университет | «Не могу сформулировать мысль по-немецки» | Академическая речь |
| «Знаю, но не говорю» | «Три года учу — и всё равно пауза на перевод» | Сломать языковой барьер |

**Главная объединяющая боль:**
> «Я знаю немецкий. Но когда надо говорить — в голове начинается перевод, поиск слов и паузы.»

### Вторичная аудитория: Русскоязычные, планирующие переезд в DACH (уровень A1–B1)

Ищут репетитора заранее. Триггер: «Хочу переехать и готовлюсь».

## 3. ПОЛНАЯ СТРУКТУРА СТРАНИЦЫ

Страница состоит из **11 секций** в следующем порядке.

```
01 HERO              — Захват + первичный CTA

03 PAIN              — Боль/Проблема: «ты меня понимаешь»
04 HOW IT WORKS      — Процесс: 3 шага от боли к результату
05 ABOUT             — Наставник: кто, откуда, почему
06 RESULTS           — Трансформации студентов: до/после
07 TESTIMONIALS      — Отзывы: 4–6 реальных историй
08 OFFER             — Диагностическая сессия: оффер
09 FAQ               — Снятие возражений: 6 вопросов
10 FINAL CTA         — Тёплое закрытие
11 FOOTER            — Контакты, правовое
```

### СЕКЦИЯ 01: HERO

**Назначение:** Захват внимания за 3 секунды. Квалификация аудитории. Один первичный CTA.

**Макет:** Split-screen (desktop) — левая колонка текст, правая фото наставника. На mobile: фото сверху, текст снизу.

**Элементы:**

```
[Лейбл-таблетка]
«Онлайн · A1–C2 · Живу в Германии»

[H1 — главный заголовок]
«Заговорите по-немецки
без паузы на перевод»

[H2 — подзаголовок]
«Индивидуальное наставничество + умная система
ежедневной практики. Для тех, кто немецкий знает,
но в нужный момент теряется.»

[Цитата — боль (сохранить из текущей версии)]
«Я понимаю немецкий, но когда надо говорить —
в голове начинается перевод, поиск слов и паузы.»

[Credibility micro-line]
«8 лет онлайн · 80+ студентов из 7 стран · Констанц, Германия»

[CTA — основной]
[Кнопка] «Записаться на диагностику» → Telegram

[CTA — вторичный, ниже кнопки]
«Диагностика бесплатна · 30 минут · Без обязательств»
```

**Фото:** Репетитор в естественной обстановке — тёплый свет, нейтральный европейский фон (кофе, книжная полка, или набережная Констанца / Бодензее). НЕ студийный хедшот. Выражение: mid-smile или mid-thought. Smart casual одежда.

**Анимации:**
- Hero заголовок — Word Swap или TypeWriter effect на ключевое слово («переводить» → «думать» → «говорить»), 2.5s цикл
- Credibility-числа — count-up при первом появлении
- Фото — fade-in с лёгким rise (0px → +24px → 0px), задержка 300ms
- CTA кнопка — pulse-glow (существующая анимация, сохранить)

### СЕКЦИЯ 02: TRUST BAR

**Назначение:** Мгновенный credibility-сигнал сразу после hero, до того как посетитель прочитал хоть что-то.

**Макет:** Горизонтальная полоса, 3–4 элемента в ряд. На mobile: 2+2 или вертикальный стек.

**Контент (выбрать реальные цифры):**

```
[Иконка ★]  «4.9 / 5»
             «средняя оценка студентов»

[Иконка 👥]  «80+ студентов»
             «из России, Украины, Казахстана, Беларуси»

[Иконка 📅]  «С 2019 года»
             «8 лет онлайн-преподавания»

[Иконка 🇩🇪]  «Живу в Констанце»
              «Германия, с [год] года»
```

**Дизайн:** Тонкая разделительная линия сверху и снизу. Числа в `--gold-light`, подписи в `--text-muted`. Минимальный, ненавязчивый блок.

**Анимации:** Count-up на числа при скролле в viewport (IntersectionObserver).

### СЕКЦИЯ 03: PAIN (Боль / Проблема)

**Назначение:** «Ты меня понимаешь» — создать глубокий эмоциональный резонанс до предложения решения.

**Заголовок секции:**
```
[Label] «Узнаёте себя?»
[H2]    «Это не вопрос знаний.
         Это вопрос мышления.»
```

**Контент — 4 сценария-монолога:**
```
→ «Понимаю немецкий фильм без субтитров.
   Но в Behörde отвечаю односложно и хочу провалиться сквозь землю.»

→ «На работе думаю по-русски, перевожу —
   и пока перевожу, собеседник уже ответил.»

→ «Знаю правило. Но в разговоре всё равно ошибаюсь —
   потому что нет автоматизма, есть только теория.»

→ «Перед экзаменом Goethe B2 всё остальное кажется простым.
   Говорение — как будто другой язык.»
```

**Мост (текст после сценариев):**
```
«Это не вопрос словарного запаса. И не вопрос старания.

Это вопрос маршрута мышления. Ваш мозг привык думать по-русски
и подключать немецкий как переводчик поверх.

Когда это меняется — меняется всё.»
```

**Дизайн:** Каждый сценарий — карточка с `border-left: 3px solid var(--gold-warm)`. Текст в кавычках курсивом. Завершающий «мост» — отдельный параграф, чуть крупнее, `--text-primary`.

**Анимации:** Staggered fade-rise при скролле. Каждая карточка с задержкой 120ms.

### СЕКЦИЯ 04: HOW IT WORKS (Как это работает)

**Назначение:** Убрать «чёрный ящик» — показать конкретно, что происходит после записи. 3 шага.

**Заголовок:**
```
[Label] «Система»
[H2]    «Три шага к речи без паузы»
[Lead]  «Не курс. Не самоучитель. Персональная система,
         которая работает каждый день — не только во время урока.»
```

**3 шага:**
```
[01] ДИАГНОСТИЧЕСКАЯ СЕССИЯ (30 мин, бесплатно)
     «Разбираем ваш уровень, цели и главный блок.
      Вы получаете конкретную картину: что именно мешает говорить
      и почему. Это полезно само по себе — даже если
      вы не продолжите со мной.»

[02] ПЕРСОНАЛЬНЫЙ ПЛАН
     «На основе диагностики составляем маршрут:
      какие конструкции отрабатывать, в каком темпе,
      в каких реальных сценариях из вашей жизни.
      Никакого универсального учебника.»

[03] ЗАНЯТИЯ + ИИ-ТРЕНАЖЁР МЕЖДУ НИМИ

      Прогресс накапливается каждый день, а не раз в неделю.»

**Split-Role блок (AI + Наставник):**

│  · Стратегию прогресса      │  · Сотни повторений         │
│  · Культурный контекст      │  · 24/7 доступность         │
│  · Эмоциональную поддержку  │  · Адаптацию под ваш уровень│
└─────────────────────────────┴─────────────────────────────┘

**Дизайн:** Нумерованный вертикальный timeline (desktop: горизонтальный). Числа `01` `02` `03` крупные, в Cormorant Garamond, `--gold-warm`. Соединительная линия между шагами «рисуется» при скролле.

**Анимации:** Staggered reveal шагов + SVG line-drawing effect для соединительной линии.

✓ Разбираем ваш реальный уровень (не по документам)

— Вы точно знаете, что вам нужно
— Вы можете работать со мной или самостоятельно
— Никаких обязательств

function countUp(el, target, duration = 1200) {
  const start = performance.now();
  const update = (time) => {
    const progress = Math.min((time - start) / duration, 1);
    const ease = 1 - Math.pow(1 - progress, 3);
    el.textContent = Math.floor(ease * target) + (el.dataset.suffix || '');
    if (progress < 1) requestAnimationFrame(update);
  };
  requestAnimationFrame(update);
}
```
